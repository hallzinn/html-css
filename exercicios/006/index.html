principais formatações

negrito / destaque

frase em negrito a seguir: para destacar um paragrafo em negrito usamos a tag 'b'. Esta tag não atribui valor semantico ao termo em destaque

ja a tag 'STRONG' atribui valor semantico e valor de forma ao termo em destaque

itálico / ênfase

usa-se a tag 'i' para os termos em italico, mas sem valor semantico

utiliza-se a tag 'EM' para os termos em italico, mas com valor semantico

marcadores de textos

cria-se textos marcados com a tag 'MARK'

para mudar a cor dentro da tag 'MARK' usa-se o atributo 'style' (CSS3)com a propriedade 'background-color'

seletores para tags

para selecionar varias tags iguais usa-se a tag 'style' na head do documento junto com a tag desejada e qual modificação atribuir

texto grande e pequeno

crianedo um texto grande e um texto pequeno neste paragrafo

texto deletado

para senalisar um texto deletado usa-se a tag 'DEL'

texto inserido

para sinalisar que um texto foi inserido usa-se a tag 'ins'

texto sublinhado

para sublinhar um texto usa-se a tag 'u' texto sublinhado (sem valor semantico)

texto sobrescrito

para sobrescrever um texto usa-se a tag 'sup' texto sobrescrito, exemplo x2

texto subscrito

para subescrever um texto usa-se a tag 'sub' texto subscrito, exemplo log2
